# coolEmployeeTracker
Uses Command Line to Read & Store Data.

## Description
Application is uses Node.js to connect to a MySQL allowing the user to store and read data from a Employee Database.

## Technologies

* Javascript
* Node.js
* MySql Workbench
* SQL
* Modularization
* Constructors
* OOP (Object-Oriented Programming)

## Packages

* Inquirer
* MySQL2
* Dotenv
* Console.table
